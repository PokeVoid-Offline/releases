Installing on Android
All Android Devices
  1. Download the APK above.
  2. Open the file — if prompted, allow your browser or file manager to install unknown apps.
  3. Tap Install and launch PokeVoid from your home screen.
Installing on iOS
SideStore + LiveContainer Recommended
  1. Install SideStore on your device.
  2. Inside SideStore, install LiveContainer.
  3. Open LiveContainer and import the IPA downloaded above.
  4. Launch PokeVoid from within LiveContainer — no Apple ID app slot consumed.
Full step-by-step guide →
AltStore
  1. Install AltStore via a PC or Mac.
  2. Open AltStore on your iPhone or iPad.
  3. Tap My Apps → + and select the downloaded IPA.
  4. Refresh every 7 days to keep it active.
Feather
  1. Install Feather using your preferred method.
  2. Tap + to import the IPA.
  3. Sign with your certificate and tap Install.
Installing on Desktop
Windows
  1. Download PokeVoidOffline.exe above.
  2. Run it — no installation needed. If SmartScreen appears, click More info → Run anyway.
Linux
  1. Download PokeVoidOffline.AppImage above.
  2. Make it executable: chmod +x PokeVoidOffline.AppImage
  3. Run it: ./PokeVoidOffline.AppImage

Installing on Android

Android requires allowing installs from unknown sources. You only need to do this once.

All Android Devices

  1. Download the APK above.
  2. Open the APK file from your downloads. If prompted, allow your browser or file manager to install unknown apps.
  3. Tap Install and launch PokeVoid from your home screen.

Installing on iOS

iOS does not allow installing apps outside the App Store without a sideloading tool. Choose the method that works best for you below.

SideStore + LiveContainer Recommended

  1. Install SideStore on your device.
  2. Inside SideStore, install LiveContainer.
  3. Open LiveContainer and import the IPA downloaded above.
  4. Launch PokeVoid from within LiveContainer — no Apple ID app limit consumed.

AltStore

  1. Install AltStore on your device via a PC/Mac.
  2. Open AltStore on your iPhone/iPad.
  3. Tap My Apps → + and select the downloaded IPA.
  4. AltStore will sign and install the app. Refresh every 7 days to keep it active.

Feather

  1. Install Feather using your preferred method.
  2. Open Feather and tap + to import the IPA.
  3. Sign the app with your certificate and tap Install.

Installing on Desktop

Windows

  1. Download PokeVoidOffline-Windows.exe above.
  2. Run the file — no installation needed. Windows may show a SmartScreen warning; click More info → Run anyway.

macOS

  1. Download PokeVoidOffline-Mac.dmg or PokeVoidOffline-Mac-arm64.dmg above.
  2. Open the DMG and drag PokeVoid Offline to your Applications folder.
  3. On first launch, macOS may block the app. Go to System Settings → Privacy & Security and click Open Anyway.

Linux

  1. Download PokeVoidOffline-Linux.AppImage above.
  2. Make it executable: chmod +x PokeVoidOffline-Linux.AppImage
  3. Run it: ./PokeVoidOffline-Linux.AppImage
